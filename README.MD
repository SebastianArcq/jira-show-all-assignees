# Jira Show All Assignees
Chrome Web Store: https://chromewebstore.google.com/detail/jira-show-all-assignees/ocanddncnmhnboilkfiihbgijabjiekd
Github: https://github.com/SebastianArcq/jira-show-all-assignees


## About
Shows all assignees in a Jira board in a single row instead of dropdown.

On a Jira board there are the avatars of the people that are working on items on this board. However, when there are more than four people on that board the rest are hidden behind a +N button, and in order to select their tickets you have to click on the +N button and then click on their avatar. This extension save you the extra click and just makes it so all the avatars are in a single row. 

That's it. Does one job and one job only.
